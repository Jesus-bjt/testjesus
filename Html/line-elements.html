Elementos de Línea

Son etiquetas que permiten personalizar de cierta forma textos

Por ejemplo:

em un enfasis
br para un salto de linea forzado en el texto
strong para mayor enfasis
small para restar le enfasis a un texto
wbr para un salto de linea en palabras largas en puntos especificos si se necesita
time para representar horas y/o fechas 2:14

Tambien esta las etiquetas sub y sup utiles en formulas quimicas y matematicas
subindicesub y superindicesup

Para personalizar el texto (a pesar que se hace en css) html tiene tres etiquetas

i para letras cursivas
b para letras con negrita
y por ultimo u para subrayar
